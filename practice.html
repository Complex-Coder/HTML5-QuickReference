Extra Markups to know about

  1. Doctypes

    2. Every page should be started with a Doctype dec;aration to tell a browser which version of HTML the page is using. For HTML5, we use <!DOCTYPE>

  2. Comments in HTML

    3. For Comments use, <!-- with some comments inside it -->

  3. ID attribute

    4. Every HTML element can carry an id attribute. No two HTML on the same page can have same id attribute. Value of an ID attribute should never start with a digit and it should only contain a letter and an underscore.

  4. CLASS attribute

    5. Every HTML element can carry a class attribute. Class of different element can have same or different values.

  5. DIV element

    6. DIV element allows us to group a set of elements together in one block-level box.
